isLiveBroadcast

Canonical URL: http://schema.org/isLiveBroadcast

Thing > Property > isLiveBroadcast

True is the broadcast is of a live event.
  • Leave public feedback on this term 💬
  • Check for open issues.
  • 'isLiveBroadcast' is mentioned in layer: core
Values expected to be one of these types
Boolean
Used on these types
BroadcastEvent


Examples

Example 1
Without Markup Microdata RDFa JSON-LD 
A broadcast of the FIFA Women's World Cup 2015 final.
